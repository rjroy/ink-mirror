# Research: Style Profile Versioning Over Time

## Question

How should the style profile capture voice evolution? If your voice changes (and it will), should old versions be preserved? What does a useful "voice diff" look like?

## Research Directions

- Linguistics research on how writing voice evolves (developmental, contextual, temporal patterns)
- Composition studies on style development in writers
- Version models: git-style snapshots, continuous deltas, periodic summaries
- Whether voice change is gradual (profile drifts) or punctuated (deliberate shifts)
- How much history is useful for self-reflection vs. noise

## Constraints

- Versioning should be automatic (frictionless principle), not require the user to "save a snapshot"
- The storage model should support meaningful comparison without overwhelming the user with diffs
